# V1.5 Release Checklist

这份 checklist 用于 `V1.5` 封板前的正式签收。  
当前版本明确不包含：

- `Company Plaza`
- `Growth / Support` 独立 bot
- `Slack`
- `L2 department promotion`
- `Quality` 双席位拆分

## 1. Runtime 与基础设施

- [ ] `python3 scripts/live_signoff.py` 自动检查通过
- [ ] `docker compose ps` 中以下服务全部正常
  - `app-dev`
  - `multi-agent-company-feishu-runner`
  - `openclaw-gateway`
  - `postgres`
  - `redis`
  - `qdrant`
  - `minio`
- [ ] `GET /api/v1/health` 返回 `status=ok`
- [ ] `GET /api/v1/openclaw/gateway/health` 返回 `status=healthy`
- [ ] Dashboard 可打开
- [ ] OpenClaw Control UI 可从 Dashboard 直开

## 2. 7 核心 Bot

- [ ] OpenClaw Control UI 与 Dashboard 都只显示相同的 7 个核心 agent
- [ ] `/api/v1/feishu/bot-apps` 返回 7 个核心 bot
- [ ] 7 个 bot 单聊全部正常
- [ ] 群聊单提及正常
- [ ] 群聊多提及正常

- [ ] 单请求总轮次不会超过 `20`

## 3. Persona / Identity / Workspace

- [ ] 7 个核心 bot 的 `SOUL.md / IDENTITY.md / AGENTS.md / BOOTSTRAP.md / SKILLS.md` 全部成功编译
- [ ] `AGENTS.md / SKILLS.md` 已进入 runtime bootstrap

  - `SOUL.md`

## 4. Skill Catalog

- [ ] 每个 bot 至少 `30` 个专业 skills

- [ ] OpenClaw Skills 页可发现各 agent 的 native skills
- [ ] 每个 skill 都有：
  - `repo_url`
  - `repo_name`
  - `commit_sha`
  - `path`
  - `license`
  - `install_method`
  - `verify_command`
- [ ] 每个 skill 都通过：
  - `source validation`
  - `install validation`
  - `invocation validation`
  - `result validation`
- [ ] `/api/v1/persona/skill-catalog/validate` 返回 `ok=true`
- [ ] Skill 安装/验证失败项为 `0`
